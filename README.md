# Vue 3 + SSR + i18n Example App

Working with vue-cli 5, webpack 5 

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### SSR
```
node server.js
```

### i18n
Default `en`, switch to `es`
